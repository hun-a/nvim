# Neo-vim settings for me!

# Prerequisites
- Install the `ripgrep` for `Telescope`
  - Mac
    ```
    $ brew install ripgrep 
    ```
  - Ubuntu
    ```
    $ sudo apt-get install -y ripgrep
    ```
- Install `Meslo LG M DZ Regular Nerd Font Complete Mono` for `nvim-tree`
  - Font files in [here](./font/)
